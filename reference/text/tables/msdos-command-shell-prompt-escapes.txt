
$PROMPT escape codes
====================

Special escape codes in the $PROMPT environment variable interpreted by COMMAND.COM

Character sequence | Displays as                                                           | Example          | Example displays as
===================|=======================================================================|==================|=========================================
$$                 | Dollar sign ( $ )                                                     | Time is $$$g     | Time is $>                               [*0] [*1]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$+                 | Zero or more consecutive plus ( + ) according to PUSHD stack depth    |                  |                                          [*0]
                   | (Command Extension)                                                   |                  | 
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$_                 | Carriage return/line feed                                             | $t$h$h$h$_$p$g   | 9:30:25                                  [*0] [*1]
                   |                                                                       |                  | C:\MYDIR>
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$a                 | Ampersand ( & )                                                       | $a               | &                                        [*0]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$b                 | Vertical slash ( | )                                                  | $p$b             | C:\MYDIR}                                [*0] [*1]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$c                 | Left parenthesis                                                      | $c               | (                                        [*0]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$d                 | Current system date                                                   | $d $n$g          | Mon 9-5-1986 C>                          [*0] [*1]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$e                 | Escape character (for use with ANSI.SYS)                              | $e[32m$n$g$e[0m  | C> (prompt is green)                     [*0] [*1]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$f                 | Right parenthesis                                                     | $f               | )                                        [*0]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$g                 | Greater than sign ( > )                                               | $p$g             | C:\MYDIR>                                [*0] [*1]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$h                 | Destructive backspace                                                 | $t$h$h$h $p$g    | 09:30:26 C:\MYDIR>                       [*0] [*1]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$l                 | Less than sign ( < )                                                  | $l$n$g           | <C>                                      [*0] [*1]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$m                 | Remote name associated with the current drive letter if network drive |                  |                                          [*0]
                   | (Command Extension)                                                   |                  | 
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$n                 | Current drive letter                                                  | Drive is $n$g    | Drive is C>                              [*0] [*1]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$p                 | Current pathname directory                                            | Path is $p$g     | Path is C:\MYDIR>                        [*0] [*1]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$q                 | Equals sign ( = )                                                     | Drive $q $n$g    | Drive = C>                               [*0] [*1]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$s                 | Space                                                                 | $p$s$g           | C:\MYDIR >                               [*0]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$t                 | Current system time                                                   | Time is $t       | Time is 09:30:25.93                      [*0] [*1]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
$v                 | DOS version number                                                    | $v               | IBM Personal Computer DOS Version   3.20 [*0] [*1]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
Any other          | Treated as character typed                                            | This is a prompt | This is a prompt                         [*0] [*1]
-------------------+-----------------------------------------------------------------------+------------------+-----------------------------------------
